rootDir and outDir

This is used cz see when you compile a TS file its JS file is generated above TS file.
by using this you create a folder and all your JS files move to a specific file....

How to work??

  1. comment-out "outDir" from tsconfig.json file
  2. "outDir" : "./fileName", - name the file



Using rootDir

To only specific folder/file to be compiled in other folder... use rootDir . It specifies which directory to compile with outDir

How to work??

  1. comment-out "rootDir" from tsconfig.json file
  2. "rootDir" : "./fileName", - name the file only which you want to compile and whose js files to be appeared in outDir file
