Topic :- Target Configuration

Link vid-22 https://youtu.be/LxccSZPTbOI

Explanation

When you compile TS files - it compiles with a specific EcmaScript Version
source : tsconfig.json file - "target": "es5" - this will be the ES-version
you can change that version

Use:
  1. when you use ES5 - it compiles your "let and const" as "Var" ,.... which looks bit older
  2. when you use ES6 and above - It will not change "let and const " to "var". It will keep as it is....
